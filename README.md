# PersonaApp

Personality matching app (React + FastAPI + PostgreSQL) that returns the top 3 idol matches from a trait-based test.

## Quick Start

```bash
./start.sh
```

Stop:

```bash
./stop.sh
```

Local URLs:
- App: http://localhost:3000
- API docs: http://localhost:8000/docs
- Health: http://localhost:8000/health
