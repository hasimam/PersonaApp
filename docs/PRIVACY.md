# Privacy

PersonaApp does not require an account or email address for result sharing.

- Result images are generated in the browser and are not uploaded to PersonaApp.
- A private link is created only after explicit confirmation.
- The link exposes a compact result snapshot only. It never includes answers, feedback ratings, database IDs, or owner credentials.
- A private link stops returning report data after 30 days. Expired records are deleted during subsequent sharing activity and maintenance cleanup, preventing unbounded storage growth without a continuously running scheduler.
- Anyone who has the private link or QR code can read the snapshot until it expires. Shared reports are read-only.
